# Evaluation Rubric - Financial Data Synthesizer

## Architecture Design (25%)

- Clear modular design
- Ability to support multiple schema formats

## Synthetic Data Quality (25%)

- Distribution realism
- Preservation of relationships

## Engineering Implementation (20%)

- Code clarity
- Use of libraries effectively

## Handling Semi-Structured Data (15%)

- JSON/XML generation logic

## Scalability and Performance (15%)

- Ability to generate large datasets
